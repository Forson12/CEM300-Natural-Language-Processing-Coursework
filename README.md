# CEM300-Natural-Language-Processing-Coursework
2113122 KOBINA AMPOFI-FORSON - CEM300 Natural Language Processing Coursework 

This project compares traditional TF-IDF clustering with SBERT sentence embeddings on a product-title dataset. K-Means, Agglomerative Clustering, and SBERT are evaluated using internal metrics to show how semantic models improve unsupervised product grouping.
